  • Design patterns

GIMM 100:
An animation class where I learned the basics of storyboarding, animation techniques, and Adobe programs including Photoshop and Animate.


GIMM 110:
A game design class were we started learning how to design basic games, especially using "Flash games" with Adobe Animate and javascript.


GIMM 200:
A follow up to GIMM 100, where we continued developing our animation techniques, and finished our music video projects.

I had no animation skill before these two classes, so please pardon the simple design, and lack of walking animation in the middle.

GIMM 250:
A continuation of skill building in animation and interactive media using Adobe Animate and javascript.
Click here to open an example of Flash media based on HTML and javascript in a new tab/window.

GIMM 270:
This class taught us how to do basic video editing in Adobe Premiere.
Topics included greenscreening, stitching, and 360 video.

GIMM 280:
280 was a class where we learned about mechanical electronics, how to use motors and electrical switches.
We also had some projects that were able to be connected to the computer and controled with digital input.

GIMM 290:
290 taught us better ways to structure game design, and how to plan out a large-scale game project.
We also leared skills with the Unity 3D program, and how to code in C# for Unity.

GIMM 300:
300 is our web design class. We overviewed HTML programming, as well as CSS and javascript for web design.
This site is a project for this class.

GIMM 310:
310 is a mobile app design class. Our class group was the first to work exclusively with iOS.

GIMM 330:
330 taught us techniques for designing 3D models. We used Maya from AutoDesk to create a low-poly model and apply basic animation to it.


GIMM 350:
350 was an advanced game design class, with more advanced skills in the Unity engine, and technical aspects of game design.
We learned how to create states for individual levels or stages, multiplayer functionality over a network, and how to implement Oculus Rift controls into our projects.

GIMM 360:
This class was all about editing audio and manipulating sound files.
We used a program called VCV Rack, which emulates a synthesizer.
I must admit, I had difficulty using this program, and do not have any projects I consider worthwhile to show.

GIMM 370:
This class is a continuation of 330, with more advanced modeling techniques.
The end goal for this class is to create a high-poly model with much better levels of detail.
